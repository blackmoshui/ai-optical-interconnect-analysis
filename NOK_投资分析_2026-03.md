# NOK（诺基亚）深度投资分析 — 2026年3月

## 概述

Nokia Corporation（NYSE: NOK，中文名"诺基亚"）是全球领先的网络基础设施和电信设备公司，总部位于芬兰埃斯波。公司业务覆盖光网络、IP路由、移动网络（5G/6G）、固网和技术授权五大领域。2025年2月以$23亿收购Infinera后，诺基亚跃升为全球第二大光传输设备供应商（仅次于华为），并拥有自研的FP5/FPcx网络处理器芯片和PSE-6s相干光学引擎。

**NOK在AI光互联产业链中的角色：**
- **光传输系统供应商**：提供从城域到长途到海底的完整DWDM光传输解决方案，800G ZR/ZR+相干可插拔模块，以及下一代1.6T/2.4T相干光学引擎
- **自研芯片双核心**：FP5网络处理器（IP路由）+ PSE-6s（相干光学DSP），两颗自研芯片构成差异化竞争壁垒
- **AI数据中心网络方案商**：7220 IXR-H6交换机（102.4Tbps）直接服务于AI数据中心Fabric网络
- **端到端网络基础设施**：从数据中心内部交换到DCI光互联到WAN骨干网，提供全距离段解决方案

**与已分析标的的关系：**
- 与 [[COHR_投资分析_2026-03|COHR]]（光模块/激光器）、[[FN_投资分析_2026-03|FN]]（光模块代工）是**上下游关系**——Nokia是光传输系统集成商，使用自研或第三方光模块
- 与 [[CIEN_投资分析_2026-03|CIEN]]（Ciena）是**最直接竞争对手**——两者在光传输和DCI市场正面交锋
- 与 [[MRVL_投资分析_2026-03|MRVL]]（Marvell）在相干光学DSP领域存在竞争——Nokia自研PSE-6s vs Marvell供应第三方
- NVIDIA $10亿投资指向AI-RAN/6G方向，与NVIDIA在AI基础设施的布局互补

**关联叙事主线：** [[二级市场投资]]——AI基础设施建设 → 数据中心互联(DCI)光传输需求爆发 → 光网络设备商直接受益；同时享受5G→6G升级周期和AI-RAN长期叙事

### FP5网络处理器——IP路由核心竞争壁垒

**为什么FP5很重要：** FP5是Nokia自研的第五代网络处理器，是7750 Service Router的"心脏"。自研芯片意味着Nokia不依赖第三方ASIC（如Broadcom），拥有完全的技术自主权和差异化能力。

| 对比维度 | Nokia FP5 | 竞争对手方案（多用Broadcom芯片组） |
|---------|----------|-------------------------------|
| **单芯片吞吐量** | 4.8Tbps | 取决于Broadcom Memory |
| **每槽位吞吐量** | 14.4-19.2Tbps | 视平台而定 |
| **800GE端口** | 首家支持48端口QSFP-DD 800 | 后续跟进 |
| **线速加密** | ANYsec（MPLS/SR线速加密） | 需额外设备或性能折损 |
| **功耗降低** | 较上代降75% | 因芯片而异 |
| **可编程性** | 完全可编程 | 部分可编程 |

**战略意义：** FP5让Nokia在高端IP路由领域拥有类似于COHR在InP激光器上的地位——自研芯片带来的成本控制和功能差异化，是非自研芯片厂商难以复制的。这也是Nokia能赢得CoreWeave、Microsoft等AI客户IP骨干网订单的技术基础。

### PSE-6s相干光学引擎——光传输技术制高点

PSE-6s是Nokia（含原Infinera技术）的第六代相干光学处理引擎，代表当前商用相干光学的最高水平之一：

| 特性 | PSE-6s | Ciena WaveLogic 6 |
|------|--------|-------------------|
| **最大传输速率** | 2.4Tbps（双通道，每通道1.2Tbps） | 1.6Tbps（单通道） |
| **波特率** | 130-140GBaud | 200GBaud |
| **工艺制程** | 5nm | 3nm |
| **800G传输距离** | >2,000km | ~1,000km+ |
| **架构差异** | 多通道集成 | 单载波极致性能 |
| **功耗** | 较上代降40% | 较上代降50% |

**技术路线图：** Nokia在OFC 2026（2026年3月）发布了下一代1.6T和2.4T相干可插拔模块，预计2027年中开始送样，2027H2量产。这些产品将针对IPoDWDM DCI和薄转发器两大场景优化。

## 综合评估与操作建议

### 估值判断

**当前估值：合理偏贵（短期），但中长期有重估空间。**

- Forward P/E ~23x在电信设备板块属于中等偏高，但在AI基础设施板块属于最便宜之一
- 股价$8.36已超过分析师共识目标价$7.00-7.30，短期追高风险存在
- 但如果AI/光网络业务持续双位数增长，市场可能给予更高估值——从"电信估值"（15-20x）重估为"AI基础设施估值"（25-30x）
- FCF收益率3.7%+股息1.3%=总股东回报~5%提供下行保护
- **关键变量：(1)AI/云客户收入占比能否从6%快速提升至15-20%；(2)光网络+IP能否维持10-12%增速；(3)移动网络是否拖累整体增速**

### 操作建议

**评级：谨慎买入 / 等待回调建仓 — AI基础设施中最便宜的标的之一，但短期已超涨。**

| 策略 | 操作 | 仓位 |
|------|------|------|
| **观望区（当前）** | $8.00-$8.80（接近52周高点，不建议追高） | 可小仓试探 |
| **首次建仓** | $7.00-$7.50（回调10-15%，接近分析师共识） | 1/3 仓位 |

| **加仓区** | $5.00-$5.50（回调35%+，Forward P/E ~15x） | 1/3 仓位 |
| **止损参考** | $4.50以下（跌破前低支撑） | 减仓/清仓 |
| **目标价（12个月）** | $9.50-$11.00 | AI重估场景 |

**建议仓位占比：** 3-5%

### 核心逻辑

**买入理由：**
1. **AI基础设施中最被低估的标的**——Forward P/E仅23x，远低于COHR(54x)、ALAB(60x+)，甚至低于CRDO(29x)，但拥有类似的AI数据中心增长叙事
2. **双自研芯片壁垒**——FP5(IP路由)+PSE-6s(相干光学)是全球独一无二的组合，技术护城河深
3. **NVIDIA战略背书**——$10亿投资+AI-RAN合作，在AI基础设施供应链中获得核心玩家认可
4. **光网络增长强劲**——Infinera收购后成为全球#2光传输供应商，光网络+17-19%增速，hyperscaler订单3倍增长
5. **正FCF+净现金+股息**——€15亿FCF、€34亿净现金、1.3%股息，提供安全边际，在高估值AI板块中难得的防御属性

**主要顾虑：**
1. **增速偏慢**——全公司仅+3%（FY2025），即使AI/光网络增长，被移动网络拖累后整体增速有限
2. **利润率偏低**——营业利润率~10%，远低于纯软件/fabless公司，限制了估值天花板
3. **短期已超涨**——$8.36超过分析师共识$7.00-7.30，追高风险收益比不佳

### 结论

> **诺基亚是AI光互联产业链中一个独特的"低估值+转型叙事"标的。其FP5网络处理器+PSE-6s相干光学引擎的双自研芯片组合在全球独一无二，NVIDIA $10亿投资和CoreWeave/Nscale等AI客户大单正在验证其从传统电信设备商向AI基础设施供应商的转型。光网络+17-19%增速和hyperscaler订单3倍增长是实打实的基本面改善。**
>
> **然而，移动网络占~39%收入但增长乏力，AI客户仅占6%集团收入，全公司增速仅+3%（FY2025），营业利润率~10%——这些"老电信"特征限制了估值上行空间。当前$8.36已超过多数分析师目标价，短期追高风险存在。建议等待回调至$7.00-7.50区间（~15%回调）首次建仓，$6.00-6.50核心建仓。12个月目标价$9.50-$11.00（AI重估场景）。在AI基础设施组合中，NOK扮演"低估值防御型AI网络基础设施仓位"角色，建议配置3-5%，与COHR(光模块)、CIEN(纯光网络)形成产业链互补。**
