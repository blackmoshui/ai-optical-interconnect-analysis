# ALAB（Astera Labs）深度投资分析 — 2026年3月

## 概述

Astera Labs（NASDAQ: ALAB）是专注于AI数据中心机架级连接的半导体公司，总部位于美国加州圣克拉拉。公司成立于2017年，由前Intel工程师Jitendra Mohan创立，2024年3月以$36/股IPO登陆纳斯达克。

**ALAB在AI数据中心中的角色：**
- **PCIe/CXL智能DSP Retimer**：Aries系列是AI服务器中GPU/加速器与CPU/Switch之间高速信号完整性的关键组件，解决PCIe 5.0/6.0高速信号衰减问题
- **智能交换芯片**：Scorpio P-Series（PCIe Gen6 Fabric Switch）和X-Series（Scale-Up Switch）是AI集群规模化的新增长引擎
- **以太网智能线缆模块**：Taurus系列提供机架内800G以太网连接
- **CXL内存控制器**：Leo系列支持CPU/GPU内存池化扩展
- **硅光领域新进入者**：2026年1月完成收购aiXscale Photonics，进入光连接芯片领域

**核心竞争优势一句话：** Astera Labs是AI数据中心"连接层"的全栈平台公司，从Retimer到交换芯片到光连接，配合COSMOS软件套件形成高粘性生态。

**与已有持仓/关注标的的关系：**
- 与 [[COHR_投资分析_2026-03|COHR]]（光模块）、[[CRDO_投资分析_2026-03|CRDO]]（AEC电互联）**互补**——ALAB做PCIe/CXL层的信号处理和交换，COHR做光传输，CRDO做短距电缆
- 与 [[MRVL_投资分析_2026-03|MRVL]]（PCIe Switch + 光DSP）和 Broadcom（PCIe Retimer + Switch）**直接竞争**
- ALAB的Aries Retimer已部署在NVIDIA HGX系统中，是NVIDIA AI服务器生态的组成部分

**关联叙事主线：** [[二级市场投资]]——AI基础设施连接层：GPU数量指数级增长 → 数据中心内部连接带宽成为瓶颈 → PCIe/CXL Retimer + 交换芯片需求爆发

### 技术深度：Scorpio Smart Fabric Switch——最大增长引擎

**为什么Scorpio至关重要：**

传统AI集群使用NVIDIA NVLink进行GPU间通信（Scale-Up），但NVLink是私有协议，只能连接有限数量的GPU（如DGX系统8颗GPU）。当需要将数百上千颗GPU组成更大集群时，需要外部Fabric Switch。Scorpio就是解决这个"Scale-Up扩展"问题的关键芯片。

| 对比维度 | NVIDIA NVLink（私有） | Scorpio X-Series（开放） |
|---------|---------------------|------------------------|
| **协议** | NVLink私有协议 | UALink/NVLink Fusion开放标准 |
| **扩展性** | 最多72-576颗GPU（NVLink Domain） | 理论上可扩展至数万颗 |
| **客户选择** | 绑定NVIDIA生态 | 支持异构加速器（NVIDIA/AMD/自研ASIC） |
| **定价** | 高（NVIDIA溢价） | Merchant Silicon更具性价比 |
| **灵活性** | 固定拓扑 | 软件可定义拓扑 |

**Scorpio P-Series vs X-Series：**
- **P-Series（已量产）**：PCIe Gen6 Fabric Switch，行业首款，用于Scale-Out PCIe连接（CPU到GPU/加速器/存储的前端连接）。FY2025已超过15%年营收
- **X-Series（初始出货）**：Scale-Up Fabric Switch，用于GPU-to-GPU后端高带宽连接，与Hyperscaler联合开发。支持增大Radix、平台专有协议、In-Network Computing、Hypercast技术、光连接。**预计2026H2材料化放量，2027年跨多客户大规模部署**

**$20B TAM：** 管理层预估Merchant Scale-Up Switching市场到2030年达$200亿。这是ALAB从"连接芯片公司"转型为"连接平台公司"的关键赌注。

### COSMOS软件平台——粘性生态

COSMOS（Connectivity System Management and Optimization Software）是ALAB区别于Broadcom/Marvell的关键差异化：

- **实时链路健康监控**：覆盖整个连接Fabric的遥测和诊断
- **预测性分析**：主动识别潜在链路故障
- **Fleet管理**：大规模机队级别的连接资源管理和优化
- **软件粘性**：一旦Hyperscaler的运维团队习惯了COSMOS，切换成本极高

**这是Astera Labs从"卖芯片"转向"卖平台"的关键——硬件竞争对手可以做类似芯片，但复制软件生态和客户运维习惯极难。**

## 综合评估与操作建议

### 估值判断

**当前估值：50%回调后进入合理区间，但仍偏贵。Forward P/E 51x需要持续60%+增速来支撑。**

- PEG 1.50合理但不便宜——对比MRVL的PEG 0.62和CRDO的PEG 0.37，ALAB的成长溢价偏高
- 但ALAB的平台化布局（Retimer + Switch + CXL + 光连接 + COSMOS软件）赋予其更宽的TAM和更高的战略价值
- Amazon $6.5B协议提供了独一无二的营收可见性
- **关键：如果Scorpio X按计划在2026H2放量，且FY2027指引达到$1.88B+，Forward P/E将降至36x——那就是真正的买入信号**
- 最大风险是毛利率持续下滑+竞争加剧导致估值进一步压缩

### 核心逻辑

**买入理由：**
1. **AI连接层全栈平台**——唯一同时覆盖PCIe Retimer + Fabric Switch + CXL + 以太网 + 光连接的专注公司，COSMOS软件构建粘性生态
2. **Scorpio打开$20B TAM**——从$8.5亿营收的Retimer公司，转型为面向$250亿SAM的连接平台公司
3. **Amazon $6.5B长期锁定**——到2033年的采购承诺提供了半导体行业罕见的营收可见性

3. **毛利率下滑趋势**——从76%→74%，Amazon Warrant继续施压

### 结论

> **Astera Labs是AI数据中心连接层最纯正的标的之一，其"Retimer→Switch→CXL→光连接"的全栈平台布局和COSMOS软件生态构建了差异化壁垒。Amazon $6.5B长期采购协议和PCIe Gen6先发优势提供了强劲的基本面支撑。**

> **然而，从$263到$132的50%暴跌虽然创造了入场机会，但客户极度集中、毛利率下滑趋势、以及Broadcom/Marvell的竞争加剧是不可忽视的风险。建议在$120-135区间试探建仓（1/4仓位），$100-115核心建仓。12个月目标价$195-250。在AI基础设施组合中，ALAB扮演"连接层高弹性卫星仓位"角色，建议配置3-5%，与CRDO（AEC）和COHR（光互联）形成互补。**
