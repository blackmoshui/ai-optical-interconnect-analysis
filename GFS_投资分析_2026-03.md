# GFS（GlobalFoundries）深度投资分析 — 2026年3月

## 概述

GlobalFoundries Inc.（NASDAQ: GFS）是全球第三大半导体代工厂（按收入），总部位于美国纽约州Malta。公司成立于2009年，由AMD的制造部门拆分而来，后与新加坡特许半导体（Chartered Semiconductor）合并，目前由阿布扎比主权财富基金穆巴达拉（Mubadala）持有约75-80%股权。

**GFS 在 AI 光互联产业链中的角色：**
- **全球最大纯硅光子代工厂**：2025年11月收购新加坡 Advanced Micro Foundry (AMF) 后，成为全球收入最大的纯硅光子代工厂
- **GF Fotonix 平台**：业内首个在300mm晶圆上整合光子特性与300GHz级RF-CMOS的单片集成平台，支持200G/λ速率，实现单芯片0.5Tbps/光纤的业界最高数据速率
- **CPO 先行者**：与 Ayar Labs、Lightmatter、Broadcom、NVIDIA、Marvell 等合作开发共封装光学 (CPO) 方案
- **差异化代工定位**：放弃先进制程竞赛（不做7nm以下），聚焦"功能丰富"节点（12nm-28nm及以上），覆盖汽车、RF、光通信、GaN等特种工艺

**与 TSEM（Tower Semiconductor）的竞争关系：** 两者是硅光子代工的双寡头——GFS 拥有300mm大晶圆制造能力和GF Fotonix单片集成优势，TSEM 拥有PH18平台的成熟量产基础和约80%的现有市占率。GFS 2025年硅光子收入 $2亿+ vs TSEM $2.28亿，差距正在缩小。两者都在激进扩产，抢夺AI光互联爆发带来的巨大增量。

**关联叙事主线：** [[二级市场投资#1. AI 基础设施]]——AI数据中心光互联从铜切换到光 → 硅光子芯片需求爆发 → 代工厂（"卖铲人"）直接受益

### GF Fotonix —— 硅光子核心竞争壁垒（深度解析）

**为什么硅光子对GFS至关重要：** 传统铜互联在AI数据中心的高带宽传输中正逼近物理极限，硅光子技术利用光信号替代电信号传输数据，实现更高速率、更低功耗、更远距离的数据传输。GFS作为纯代工厂，为所有fabless光芯片设计公司制造硅光子芯片——这是典型的"卖铲人"角色。

| 对比维度 | GF Fotonix（GFS） | PH18平台（TSEM） |
|---------|-------------------|-----------------|
| **晶圆尺寸** | **300mm（12英寸）** | 200mm（8英寸）为主 |
| **单片集成** | **光子+RF-CMOS+逻辑** | 光子为主，需外挂电芯片 |
| **最高速率** | **200G/λ**（0.5Tbps/fiber） | 100G/λ（PH18HA） |
| **路线图** | 400G/λ开发中 | 200G/λ开发中 |
| **CPO支持** | 原生支持可拆卸互联 | 支持 |
| **2025年SiPh收入** | ~$2亿+ | ~$2.28亿 |
| **2026年SiPh收入目标** | ~$4亿（翻倍） | ~$4.5亿+（翻倍） |
| **2028年目标** | $10亿年化 | — |
| **市占率** | ~30%（追赶中） | ~80%（领导者） |

**GF Fotonix的独特优势：**
1. **300mm大晶圆**：相比TSEM的200mm平台，单晶圆产出更多芯片，规模化后成本优势显著
2. **单片集成**：光子引擎+驱动电路+逻辑控制在同一芯片上——减少封装复杂度、降低寄生效应、提升性能
3. **200G/λ领先**：当前已量产200G/λ，支持1.6T/3.2T光模块需求
4. **CPO原生架构**：在OFC 2026展示可拆卸光互联技术，专为CPO设计

## 赛道分析

### 行业定位与产业链角色

GFS在AI光互联产业链中扮演**上游代工**角色——为所有硅光子芯片设计公司提供制造服务：

| 产业链层级 | 角色 | 代表公司 |
|----------|------|---------|
| **芯片设计（Fabless）** | 设计硅光子芯片 | Ayar Labs, Lightmatter, Broadcom, Marvell, NVIDIA |
| **代工制造（Foundry）** | **制造硅光子芯片** | **GFS, TSEM, TSMC（少量）** |
| **光模块封装** | 封装成光收发器 | COHR, LITE, FN（代工） |
| **系统集成** | 数据中心网络设备 | ANET, CSCO, NOK |
| **终端用户** | 部署AI集群 | GOOG, META, MSFT, AMZN |

**核心判断：** GFS是光互联产业链中最"卖铲"的角色——无论哪家光芯片设计公司获胜，都需要代工厂制造芯片。这种"不论谁赢都受益"的商业模式，类似于淘金热中卖铲子和牛仔裤的角色。

### 行业趋势与驱动力

1. **AI集群光互联需求爆发**：从万颗GPU到百万颗GPU，每个GPU需要更多光连接，硅光子芯片需求指数级增长
2. **铜到光的切换**：AI数据中心从短距铜缆向光互联迁移，硅光子是核心使能技术
3. **速率升级**：800G→1.6T→3.2T，每代升级需要更先进的硅光子芯片
4. **CPO商业化**：共封装光学将光子直接放在芯片旁，2027-2028大规模部署
5. **地缘政治驱动在岸制造**：CHIPS Act推动美国本土代工，GFS作为美国最大特种代工厂直接受益

## 综合评估与操作建议

### 估值判断

**当前估值：合理偏低，硅光子叙事尚未充分定价，但整体增速有限。**

- Forward P/E ~24x + EV/EBITDA ~9.8x在半导体中算偏低估值，反映市场对代工行业增速的谨慎预期
- 但硅光子业务从$2亿增长到$10亿的路径如果兑现，将为GFS注入高增长基因，改变估值叙事
- Mubadala减持造成的持续供给压力是股价"锚"——短期压制估值扩张
- $40亿现金+净现金状态提供了安全边际
- **核心变量：硅光子能否成为增长引擎（目前仅占~3%收入），以及代工行业何时迎来真正的上行周期**

### 操作建议

**评级：谨慎买入 / 逢Mubadala减持引发的暴跌买入**

| 策略 | 操作 | 仓位 |
|------|------|------|
| **试探/首次建仓** | $38-$42（Mubadala减持后的低位区） | 1/4 仓位 |
| **核心建仓** | $34-$38（回调20%+或行业恐慌） | 1/3 仓位 |
| **加仓区** | $30-$34（接近52周低点区间） | 1/4 仓位 |
| **止损参考** | $28 以下（跌破52周低点，基本面恶化） | 清仓/减仓 |
| **目标价（12个月）** | $55-$60 | 接近分析师最高目标 |

**建议仓位占比：** 3-5%（中等仓位，波动可控的AI光互联"卖铲"标的）

### 核心逻辑

**买入理由：**
1. **AI光互联"终极卖铲人"**——全球最大纯硅光子代工厂，客户覆盖Ayar Labs/Lightmatter/Broadcom/NVIDIA/Marvell，不论谁赢都受益
2. **GF Fotonix 300mm技术领先**——业界唯一300mm单片光子+RF-CMOS集成平台，200G/λ速率领先TSEM的100G/λ
3. **硅光子收入高速增长**——2025翻倍至$2亿+，2026目标再翻倍至$4亿，2028目标$10亿年化，CAGR ~70%
4. **估值安全边际**——Forward P/E ~24x + EV/EBITDA 9.8x + $40亿净现金，估值不贵且有安全边际
5. **地缘政治受益者**——美国最大特种代工厂+$15亿CHIPS Act资金+Trusted Foundry认证，在"去中国化"趋势中直接受益

**主要顾虑：**
1. **Mubadala持续减持**——每次减持造成5-10%暴跌，是GFS独有的"结构性利空"
2. **整体增速有限**——全公司营收增速仅+6%，硅光子目前仅占~3%收入，无法单独驱动估值重估
3. **TSEM在硅光子的领先优势**——80%市占率+70%产能预订到2028，GFS追赶难度不小

### 结论

> **GlobalFoundries是AI光互联产业链中最具"卖铲人"属性的标的——作为全球最大纯硅光子代工厂，无论哪家光芯片设计公司获胜都需要GFS的制造服务。GF Fotonix平台的300mm单片集成和200G/λ速率提供了技术差异化，硅光子收入从$2亿到$10亿的增长路径清晰。然而，整体代工业务增速平缓（+6%），硅光子目前仅占3%收入，加上Mubadala持续减持的供给压力，股价难以出现COHR式的爆发行情。**
>
> **建议策略：利用Mubadala减持造成的暴跌作为建仓窗口，$38-42区间试探建仓，$34-38核心建仓。12个月目标价$55-60。在AI光互联组合中，GFS扮演"低估值卖铲仓位"角色，与TSEM形成硅光子代工双配置，与COHR/FN形成产业链纵向互补。建议仓位3-5%。**
